# Список дел

Одностраничное приложение на Vite + Vue 3 для планирования задач по четырём периодам: сегодня, завтра, вчера и через год. Задачи можно добавлять через форму, после чего они автоматически попадают в нужную колонку.

## Возможности
- Группировка дел по выбранному периоду.
- Быстрая форма добавления с полями «Название», «Описание» и «Когда выполнить».
- Отображение карточек задач с поясняющим бейджем периода.

## Требования
- Node.js 18+ (проект собирался на Node 24).

## Установка и запуск
```bash
npm install
npm run dev
```
Откройте ссылку из консоли (обычно `http://localhost:5173`).

## Сборка
```bash
npm run build
```
Готовый билд появится в директории `dist/`.
